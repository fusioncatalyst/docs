FusionCatalyst is a comprehensive platform for building event-driven, schema-first applications. With the Paw CLI, you can manage projects, define schemas, and generate type-safe code across multiple programming languages.

## Getting Started

### 1. Install the Paw CLI

```bash
# Download the latest release
curl -L https://github.com/fusioncatalyst/paw/releases/latest/download/paw_$(uname -s)_$(uname -m).tar.gz | tar xz

# Move to your PATH
sudo mv paw /usr/local/bin/
```

### 2. Initialize Your Project

```bash
# Create settings file
paw init-settings-file --language typescript

# Authenticate
paw auth signin --email your@email.com --password yourpassword --save-token

# Create a new project
paw projects new --name "My API" --belongs-to user
```

### 3. Define Your Schemas

```bash
# Create a schema
paw schemas new --project-id "your-project-id" \
  --name "User" \
  --type jsonschema \
  --schema-file ./schemas/user.json
```

### 4. Generate Code

```bash
# Create an app
paw apps new --project-id "your-project-id" --name "Backend API"

# Generate TypeScript code
paw codegen app --app-id "your-app-id" --language typescript
```
